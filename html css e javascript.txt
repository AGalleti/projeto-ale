Analogamente ao corpo humano, o desenvolvimento web se baseia em três elementos essenciais: HTML, CSS e 
JavaScript, cada um desempenhando um papel fundamental na criação de experiências online.

HTML pode ser comparado ao esqueleto humano, constituindo a estrutura de base que sustenta todo o 
conteúdo de uma página web. Assim como os ossos fornecem suporte ao corpo humano, o HTML fornece a 
estrutura fundamental para o conteúdo digital.

CSS, por sua vez, representa a pele, cabelo e roupas que envolvem o corpo humano, criando o visual e a 
aparência de uma página da web. É responsável por dar estilo e personalidade ao conteúdo, tornando-o 
agradável visualmente.

Finalmente, JavaScript atua como o músculo, conferindo movimento e interatividade ao site. Da mesma 
forma que os músculos permitem que o corpo se mova, o JavaScript possibilita a criação de funcionalidades 
dinâmicas e interativas em uma página web.
